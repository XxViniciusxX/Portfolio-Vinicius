🔐 Sistema de Segurança Inteligente (C++)

Sistema de segurança inteligente desenvolvido em C++, utilizando OpenCV e técnicas de Visão Computacional.

O projeto tem como objetivo detectar rostos em tempo real através da câmera, registrar eventos de presença e servir como base para futuras funcionalidades como reconhecimento facial e controle de acesso.

📌 Funcionalidades

  • 📷 Captura de vídeo em tempo real
  • 🧠 Detecção facial com OpenCV
  • 🟩 Bounding boxes
  • 🧾 Registro de eventos em log
  • 🔄 Controle de entrada e saída
  • 🧱 Código modular em C++

🛠 Tecnologias

C++17 | OpenCV | CMake | ONNX | Git | Visual Studio

🛡️ Scanner de Malware em Python

Sistema desenvolvido para análise de arquivos em busca de ameaças, utilizando hashing SHA256 e comparação com base de dados maliciosa.

📌 Funcionalidades

  • 🔍 Varredura de pastas e subpastas
  • 🧠 Cálculo de hash SHA256
  • ⚠️ Identificação de arquivos maliciosos
  • 📊 Relatório de análise

💡 Ideia do projeto

Desenvolvido com foco em aprendizado de segurança ofensiva e defensiva, simulando o funcionamento básico de antivírus.

🛠 Tecnologias

Python | Hashing | Segurança de Sistemas
